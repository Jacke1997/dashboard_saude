# Dashboard para Gestão em Saúde

Sistema para controle de consultas médicas com visualização financeira.

## Como executar

Instalar dependências:
pip install matplotlib

Popular os dados de exemplo:
python seed_data.py

Rodar o sistema:
python main.py

## Arquivos

- main.py — menu principal
- models.py — estrutura dos dados
- storage.py — leitura e gravação
- services.py — cadastro e busca
- rules.py — cálculos e KPIs
- dashboard.py — gráficos e relatórios
- seed_data.py — dados de exemplo
